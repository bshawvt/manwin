# A very simple window manager hobby project for the X window system.

To build: 

```gcc main.c -o3 -lX11 -o winman```

Current hotkeys:

    Move window: Super + Mouse 1
    Resize window: Super + Mouse 2
    Activate window: Super + F1
    Close window: Super + F4
